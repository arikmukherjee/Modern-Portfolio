## Overview

This repository contains a single-page portfolio website built with **HTML**, **Tailwind CSS**, and **JavaScript**. It includes sections for:

- Hero introduction
- About me
- Skills and technology proficiency
- Featured projects with live demos and GitHub links
- Experience timeline
- Contact information and form

## Highlights

- **Modern responsive design** with dark mode support
- **Interactive UI** with smooth scrolling, typing animation, and section reveal effects
- **Developer portfolio** identity for internships and job opportunities
- **Resume download** and contact call-to-action
- **Clean single-file deployment** with no build step required
